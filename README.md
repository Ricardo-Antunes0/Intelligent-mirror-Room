# Requirements
We advice you to use a virtual environment.

### Raspberry Pi 4
```bash
$ pip3 install -r requirements.txt
``` 

# To run
Turn on the Magic mirror (on the Magic/ directory)
```bash
$ npm start
```
Turn on admin page ( on Magic/Webapp_alerts)
```bash
$ python3 app.py
```
