# Requirements
We advice you to use a virtual environment.

### Raspberry Pi 4
```bash
$ pip3 install -r requirements.txt
``` 

# To run
Turn on the React Server (on the react-webapp/ directory, port 3000)
```bash
$ npm start
```
Turn on the python3-flask server
```bash
$ python3 mqtt-sub.py
```
